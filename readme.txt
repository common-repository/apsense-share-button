Help visitors share your posts on the business social network - APSense.com.

== Description ==

Add simple social sharing buttons to your articles. Your visitors will be able to easily share your content on the global business social network - APSense.com. 

**Main features / advantages**

* Simple and easy to use.
* Compatible with all major web browsers: Firefox, Chrome, Internet Explorer, Safari, Opera.
* Hide share buttons for specific pages.
* Added APSense referral link.


== Installation ==

1. Download plugin file (.zip)
2. Extract zip file and upload folder /apsense-share-button/ to /wp-content/plugins/
3. Go to "Plugins" and activate the plugin


== Frequently Asked Questions ==

= How do I change settings? =

From the Wordpress Administration go to "Plugins" and click on "Settings" under plugin name

= Are share buttons using Javascript? =

Yes. Javascript is required for sharing buttons to function properly and it must be enabled.
